2020 Q4: Program Launched. Core product Feasibility Research. Core team building.

2021 Q2: V1 online. Start to test core functions like minting synthetic assets, trading, and lending

2021 Q3: Contract code audit. Start Play to Earn model. Begin Beta test.

2021 Q4: First Batch of NFT card pre-order. Upgrade, Crafting, Farming and GameFi will be available.
